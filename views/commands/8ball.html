!8ball

The !8ball command returns a random '8-Ball' type of message

Usage

!8ball question

Example

!8ball will I win the lottery?

Could return:

@ygtskedog -> It is certain.

Note: The default permission level for !8ball is set to everyone.

  • 'Better not tell you now.'

  • 'Most likely.'
