Default Commands

  • !8ball - Returns a random '8-Ball' type of message
  • !bf4stats username platform (pc/xbox/ps3/xone/ps4) - Returns username's rank, k/d and full stats link
  • !blacklist - Returns a link to the song blacklist for the channel
    • add song - Adds song to the blacklist
    • add current - Adds the current playing song to the blacklist and skips the song
    • delete song - Removes song from the blacklist
  • !commands - Returns a link to the user added commands for the channel
    • add !command text - Adds !command to the channel
    • addalias !newcommand !commandtoalias - Adds !newcommand as an alias for !commandtoalias
    • edit !command command text - Edits !command to contain command text
    • permissions !command 0 - 4 - Sets the needed permission level for the given !command
    • delete !command - Removes !command
  • !currentsong - Returns the currently playing song
  • !firstseen username - Returns when username was first seen in the channel
  • !followage username - Returns how long username has been following the channel
  • !game - Returns the game currently being played on the channel
    • newgame - Updates the current game on Twith to newgame
  • !lastseen username - Returns when username was first seen in the channel
  • !lastsong - Returns the last song that played
  • !pause - Pauses the music
  • !play - Plays the music
  • !playlistrequest playlist - Allows users to request x amount of songs from a YouTube playlist. Can be a URL or playlistID.
  • !promote queue position - Promotes queue position to be the next song played
  • !regulars - Used to add and remove regulars from the channel
    • add username - Add username as a regular for the channel
    • delete username - Deletes username as a regular for the channel
  • !removesongs queue position or username: Removes song by queue position or all songs by username.
  • !shuffle - Shuffles all songs in the song queue
  • !skipsong - Skips the currently playing song
  • !songcache - Keeps a record of all songs that have been requested, and doesn't allow the same songs to be repeated within x time frame.
  • !songlist - Returns the list of songs in the queue
  • !songrequest song - Adds the song to the song list. Can be a URL, videoID, or song title.
  • !title - Returns the current title of the stream
    • newtitle - Updates the current title of the stream on Twitch to newtitle
  • !uptime - Returns the uptime
  • !viewers - Returns the current number of viewers
  • !volume - Returns the volume for the currently playing song
  • !winner - Returns a random user from the viewer list as a 'winner'
  • !wrongsong - Removes the last song requested by the user calling the command.
