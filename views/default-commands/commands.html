!commands

The !commands command returns a link to the user added commands for the channel and allows you and your moderators to add, edit, and remove commands.

Adding Commands

Usage:

!commands add !command text

!command is the name of the command you wish to add.

text is the message you want the bot to reply with when the command is called.

Example:

!commands add !testcommand This is a test!

Would return:

@YgTSkedog -> The command !testcommand has been added!

Adding Alias Commands

Usage:

!commands addalias !newcommand !commandtoalias

!newcommand is the name of the command you wish to add as an alias.

!commandtoalias is the name of the command you are making an alias for.

Example:

!commands addalias !testalias !testcommand

Would return:

@YgTSkedog -> The alias command !testalias has been added!

Editing Commands

Usage:

!commands edit !command text

!command is the name of the command you wish to edit.

text is the message you want the bot to reply with when the command is called.

Example:

!commands edit !testcommand This is the edited command!

Would return:

@YgTSkedog -> The command !testcommand has been added!

Setting Command Permissions

Usage:

!commands permissions !command permissionLevel

!commands permission !command permissionLevel

!commands perms !command permissionLevel

!command is the name of the command you wish to set permissions for.

permissionLevel is the permission level you want to set.

  • everyone
  • regulars
  • subscribers
  • moderators
  • owner

Example:

!commands permissions !testcommand moderators

Would return:

@YgTSkedog -> The command !testcommand permissions have been updated!

Deleting Commands

Usage:

!commands delete !command

!command is the name of the command you wish to delete.

Example:

!commands delete !testcommand

!commands remove !testcommand

Would return:

@YgTSkedog -> The command !testcommand has been deleted!

Note: The default permission level for !commands is set to everyone. However the permission level for the ability to add, edit, or delete commands is set to mods and above
