!bf4stats

The !bf4stats command returns a players rank, kill/death ratio, and a link to more stats. Platform options are pc, xbox, ps3, xone, and ps4.

Usage

!bf4stats playername platform

Example

!bf4stats twitchygtskedog pc

Would return:

@YgTSkedog -> TwitchYgTSkedog has played ~890 hours, has a 2.38 k/d, and is rank 140! More stats here: https://bf4stats.com/pc/TwitchYgTSkedog

Note: The default permission level for !bf4stats is set to everyone. If no platform is specified, it defaults to pc.
